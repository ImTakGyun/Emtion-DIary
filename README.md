## 실행 방법

1. 레포지토리를 클론하신 뒤 아래 명령어를 입력하여 dependencies를 설치 해 주세요

```
npm i
```

2. 모든 dependencies의 설치를 확인하신 뒤 다음 명령어를 입력하여 React Application을 실행하세요

```
npm run start
```
